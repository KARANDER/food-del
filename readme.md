# Full-Stack Food Delivery Application 🍔🚀

This project is a full-stack food delivery application with a backend built using **Node.js, Express, and MongoDB**, and a frontend built using **React and Vite**. The project is divided into three main parts: the **backend**, the **frontend**, and the **admin panel**.

## Backend 🛠️
The backend is responsible for handling API requests, managing the database, and providing authentication and authorization. It is built using **Node.js, Express, and MongoDB**.


## Frontend 🎨
The frontend is a **React** application built using **Vite**. It provides the UI for customers to browse food items, add items to the cart, and place orders.

## Admin Panel 🛠️
The **admin panel** is a separate **React** application built using **Vite**. It allows administrators to manage food items and orders.

## Running the Project ▶️
To run the project, follow these steps:

### 1️⃣ Start the Backend Server:
```bash
cd backend
npm install
npm run server
```
Backend server will be available at: **http://localhost:4000**

### 2️⃣ Start the Frontend:
```bash
cd frontend
npm install
npm run dev
```
Frontend will be available at: **http://localhost:5173**

### 3️⃣ Start the Admin Panel:
```bash
cd admin
npm install
npm run dev
```
Admin panel will be available at: **http://localhost:5174**
